What you will learn

  • Define data collection as a systematic, objective-oriented process that precedes dataset curation
  • Distinguish primary, secondary, and real-time data sources and their trade-offs
  • Match surveys, observation, web scraping, and APIs to research questions
  • Compare manual and automated execution for the same collection method
  • Apply probability and non-probability sampling designs appropriately
  • Select survey, scraping, and API tools that implement earlier method choices
  • Anticipate quality, bias, and integration challenges during collection design
  • Recognize how collection patterns appear across social, climate, and healthcare domains
