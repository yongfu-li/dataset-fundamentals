# IAA calculator

Interactive browser tool for **Chapter 4 — Data Annotation** (§4.5.2).
Load two annotators' labels, compute Cohen's κ (and optional Fleiss' κ with a
third rater), inspect the contingency table and disagreements, and export an
agreement report.

Runs entirely client-side — works on GitHub Pages and from `file://`
(classic IIFEs on `window.IaaLib`; presets in `data/presets-bundle.js`).

**Live path:** `lectures/tools/iaa/index.html` (after `build_site.py`)

## Learning objectives

- Treat inter-annotator agreement as a **measurable gate**, not a vibe (§4.5.2).
- Compute **Cohen's κ** from Po and Pe: κ = (Po − Pe) / (1 − Pe).
- Read a **contingency table** and list of disagreements.
- Recognize that **low κ** (e.g. ≈ 0.35 in `eg:4.31`) should stop scale-up until guidelines are revised.
- Optionally compute **Fleiss' κ** when three raters label the same items (Table 4.4).

## Workflow

1. **Load data** — `ner-org-pilot`, `sentiment-calibrated`, or `toxicity-3class`, or upload CSV/JSON.
2. **Map columns** — rater A, rater B; optional ID, text, rater C.
3. **Read meters** — κ, Po, Pe, percent agreement (hover ⓘ for formulas).
4. **Inspect contingency** — diagonal = agreements.
5. **Edit disagreements** — flip a label and watch κ update.
6. **Export** — `agreement-report.json`.

## Bundled presets

| Preset | Teaching focus |
|--------|----------------|
| `ner-org-pilot` | κ ≈ 0.35 on ORG vs O — stop and revise (eg:4.31) |
| `sentiment-calibrated` | High κ after calibration — ready to scale |
| `toxicity-3class` | Three raters for Fleiss κ |

Regenerate with `python data/make_presets.py`.
