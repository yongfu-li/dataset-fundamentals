## Learning objective

## Chapter context

## What this example shows

Following Example 3.24, governance analyses tie cybersecurity oversight failures to a reduced sale price in the Verizon acquisition—breach handling affects enterprise valuation and board accountability.

## Key terms

- **Cyber governance** — Board-level oversight of security risk and disclosure.
- **Acquisition discount** — Price adjustment reflecting breach liability and trust loss.
- **Board accountability** — Director duties when incidents overlap with strategic transactions.

## What you should learn

### From the concept
- Security is an M&A due-diligence variable.
- Late disclosure erodes negotiating leverage.
- Oversight failures are governance failures, not only SOC misses.
