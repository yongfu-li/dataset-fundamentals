## What this example shows

If the data follows a normal distribution with a mean of 50 and a standard deviation of 10, synthetic data points can be generated by sampling from a normal distribution with the same parameters.

## Key terms

- **Monte Carlo simulation** — Repeated random sampling to approximate an outcome distribution.

## What you should learn

### From the code / process
- Draw `random.gauss(50, 10)` samples and compare sample mean/sd to targets.
- Fixed seed makes output reproducible for teaching.
- Parametric sampling copies moments, not individual real rows.

### From the output / result

## How to interpret the result

If sample mean and sd drift far from 50 and 10 with large n, the generator is mis-specified—joint models (Example 10.8) add constraints univariate sampling lacks.
